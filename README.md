📊 Online Voting Application (Spring Boot + Angular)

A full-stack Online Poll / Voting Application where users can create polls and vote on different options in real-time.

🚀 Features
📝 Create new polls with multiple options
📊 View all available polls
🗳️ Vote on poll options
🔄 Real-time vote count updates
💾 Persistent storage using MySQL
🛠️ Tech Stack
🔹 Backend
☕ Java + Spring Boot
📦 Spring Data JPA
🗄️ MySQL
🔁 REST APIs
🔹 Frontend
🅰️ Angular
🎨 Bootstrap
🔗 HTTPClient (API integration)
